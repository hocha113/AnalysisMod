This is a new analysis case Mod, equivalent to the Chinese version of ExampleMod. 
If you are a Mod maker whose native language is Chinese, reading the source code of this Mod to learn TmodLoader module development is a good choice!
The AnalysisMod adds Chinese translations and numerous additional annotations to all comments based on ExampleMod, and extends more additional parsing cases!

After each update, you need to extract it again in order to obtain the latest version of the AnalysisMod's source code. 
Alternatively, you can visit the GitHub page to extract the latest source code for AnalysisMod:
https://github.com/hocha113/AnalysisMod

This Mod is only for developers' communication and learning purposes. Please refer to the ExampleMod source code at:
https://github.com/tModLoader/tModLoader/tree/1.4.4/ExampleMod

Regarding the update of version 1.3.2:
1- Added: AdditionalAnalysis section
2- Fixed incorrect images and some spelling errors
