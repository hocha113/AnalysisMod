This is a new analysis case module, equivalent to the Chinese version of ExampleMod. If you are a mod maker whose native language is Chinese, 
reading the source code of this module to learn TmodLoader mod development is a good choice!
AnalysisMod adds Chinese translations and numerous additional annotations to all comments based on ExampleMod, and extends more additional parsing cases!
This module is only for developers' communication and learning purposes. For ExampleMod source code, please visit: https://github.com/tModLoader/tModLoader/tree/1.4.4/ExampleMod
After each update, you need to extract it again to get the latest version of AnalysisMod's source code. 
Alternatively, you can go to GitHup to extract the latest AnalysisMod source code: https://github.com/hocha113/AnalysisMod

Regarding the 1.2 version update:
1-Resetting item, Buffs, block and NPC textures for better distinction from Example Mod and easier demonstration.
2-Improved translations throughout and added correct Chinese copies for annotations in AnalysisCommon/Players folder.
